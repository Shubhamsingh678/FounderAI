# FounderAI

FounderAI is an AI-powered startup planning platform.

## Features

- Startup Idea Generator
- Business Model Canvas
- AI Co-Founder Chat
- Competitor Analysis
- Marketing Plan Generator
- Pitch Deck Creator

## Tech Stack

- React
- Tailwind CSS
- PostgreSQL
- Gemini AI
- Cloud Authentication

## Purpose

FounderAI helps entrepreneurs transform ideas into business plans using Generative AI.
